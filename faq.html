Frequently Asked Questions

FAQ

What is Mob Control?

Mob Control is a mobile controller that aims to turn smartphones into responsive game controllers for interacting with desktop or console-based games.

Which platforms are supported?

Mob Control currently only supports Windows 10 and above for the desktop application and Android 8+ for the mobile application. Additional platforms will be considered for future updates.

Do all devices need to be on the same network?

Yes. For local multiplayer and low-latency input, all connected devices must be on the same Wi-Fi network or else there may be connectivity issues.

Is Mob Control safe to use?

Rest assured that Mob Control is a legal product that aims to provide accessibility to all users. We do not support any forms of data and privacy infringement.

Is Mob Control free to use?

Mob Control is free. However, there are advanced features that requires a premium subscription.

Does Mob Control require high system resources?

No. Mob Control is designed to be lightweight and efficient.
