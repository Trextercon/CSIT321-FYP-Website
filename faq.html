Frequently Asked Questions

Answers to common questions about Mob Control, setup, and supported features.

What is Mob Control?

Mob Control is a mobile controller emulator that allows users to turn their smartphones into responsive game controllers for supported platforms.

Which platforms are supported?

Mob Control supports compatible desktop games and emulator-based platforms. Additional platforms may be added in future updates.

Do all devices need to be on the same network?

Yes. For local multiplayer and low-latency input, all connected devices must be on the same Wi-Fi network.

Is Mob Control free to use?

Mob Control is currently provided as a free project version. Feature availability may change in future releases.

Can multiple players connect at the same time?

Yes. Mob Control supports multiple mobile devices connecting simultaneously for local multiplayer gameplay.
